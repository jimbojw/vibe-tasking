## Separator Examples within H1 + Inline Code

**Context:** Displaying "Held Dice" then "Remaining Dice to Roll".
**Example Dice:** Held `⚄ ⚄ ⚄ ⚀`, Remaining `⚁ ⚂`

### Using Pipe `|` (ASCII)

# `⚄ ⚄ ⚄ ⚀ | ⚁ ⚂`

### Using Light Vertical `│` (U+2502)

# `⚄ ⚄ ⚄ ⚀ │ ⚁ ⚂`

### Using Heavy Vertical `┃` (U+2503)

# `⚄ ⚄ ⚄ ⚀ ┃ ⚁ ⚂`

### Using Double Vertical `║` (U+2551)

# `⚄ ⚄ ⚄ ⚀ ║ ⚁ ⚂`

### Using Em Dash `—` (U+2014)

# `⚄ ⚄ ⚄ ⚀ — ⚁ ⚂`
