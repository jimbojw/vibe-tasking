# AI-Guides

This directory is for storing AI-Guides specific to this project. AI-Guides provide AI assistants with conceptual understanding, operational procedures, or decision-making frameworks relevant to this project's unique needs and conventions.

You can organize guides directly in this folder or create subdirectories for better organization, especially if a guide has associated template files.

For general information on Vibe Tasking AI-Guides, or for assistance in creating or using AI-Guides specific to this project, please start a new chat session by referencing your project's `CONTEXT.md` (e.g., by typing `@CONTEXT.md`) and ask your AI assistant for guidance.
